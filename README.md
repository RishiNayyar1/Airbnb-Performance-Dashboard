# Airbnb-Performance-Dashboard

# Project Overview

This project presents an interactive Power BI dashboard built to analyze Airbnb listing performance, market trends, host behavior, and customer ratings across multiple cities and years. The dashboard transforms raw Airbnb listing data into actionable insights through data modeling, DAX calculations, and interactive visualizations.

# Objectives
Analyze Airbnb listing growth trends over time.
Identify top-performing cities and room types.
Evaluate host performance and Superhost contributions.
Understand customer rating patterns across different categories.
Perform market share and Pareto analysis to identify key revenue and listing contributors.

# Key Features
Interactive Power BI dashboard with drill-down and filtering capabilities.
KPI tracking for listings, hosts, ratings, and room types.
Market share analysis by city using DAX measures.
Pareto analysis with cumulative percentage calculations.
Trend analysis of new listings across Airbnb lifecycle stages.
Room type distribution and performance comparison.
Host and Superhost performance analysis.
Dynamic visualizations and custom dashboard design.
